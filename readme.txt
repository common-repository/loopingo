Add loopingo onto the thank you page of WooCommerce to show relevant incentives for your customers.



== Description ==

Create added value for your customers while earning additional profits.
With loopingo Monetise you offer your customers exclusive vouchers after the purchase.

Your customers are looking forward to relevant offers from our network - and you earn with loopingo.

= What it does? =
The loopingo Plugin adds the code to the thank you page on WooCommerce and creates a settings page, where it can be switched on/off as well as switched between live and test mode. Additionally the Token can also easily be added.

= Features =
* Add settings page to make loopingo easy to install
* Automatically add the code to the Thank You page


= System Requirements =
* PHP >=5.6
* WordPress >=4.6
